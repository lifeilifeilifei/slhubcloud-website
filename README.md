# SLHubCloud Website

SLHubCloud 企业官网前端项目，基于 `Vite + React + TypeScript` 构建，可直接部署为静态站点。

## 本地开发

```bash
npm install
npm run dev
```

默认开发地址：

```text
http://localhost:4173
```

## 生产构建

```bash
npm run build
```

构建完成后，部署 `dist/` 目录即可。

## 可直接接入的部署平台

- Vercel
- Netlify
- Nginx 静态站点
- 对象存储 + CDN

## 上线前建议替换的信息

- 公司真实邮箱与联系电话
- 公司介绍文案
- 客户案例与业务数据
- 正式域名 `https://www.slhubcloud.com`
